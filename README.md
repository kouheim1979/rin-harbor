# リンハーバー

リンちゃん専用の、港を舞台にした合成パズルゲームです。

## 内容

- 単一HTMLアプリ
- 外部ライブラリなし
- 外部通信なし
- スマホ対応
- localStorageによる自動保存
- 合成ボード
- 注文達成
- 星によるストーリー解放
- リン図鑑

## 遊び方

1. `index.html` をブラウザで開く
2. 「リンカフェ屋台」や「海辺の箱」をタップする
3. 同じアイテムを2つ選んで合成する
4. 注文に必要なアイテムをそろえる
5. 納品して星を集める
6. 星でリンハーバーの物語を進める

## GitHub Pagesで公開する方法

GitHubのリポジトリ画面で以下を設定します。

1. `Settings` を開く
2. 左メニューの `Pages` を開く
3. `Build and deployment` の `Source` を `Deploy from a branch` にする
4. `Branch` を `main`、フォルダを `/root` にする
5. `Save` を押す

数分後、以下のURLで開けるようになります。

```text
https://kouheim1979.github.io/rin-harbor/
```
